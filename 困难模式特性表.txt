【难度定位】
  高压力模式，模拟真实的学业焦虑。每次移动扣精神/成绩，
  成绩增益缩水，假期门槛更高，精神崩溃时触发额外惩罚。

─────────────────────────────────────
一、成绩衰减（applyGradeDecay）
─────────────────────────────────────
  每步移动自动触发（从第1张图就开始）：

  ┌─────────────┬──────────┬──────────┐
  │   进度阶段    │  成绩水平  │  精神状态  │
  ├─────────────┼──────────┼──────────┤
  │  第5-6张图   │   -4     │    -5    │
  │  第3-4张图   │   -3     │    -3    │
  │  第1-2张图   │   -1     │    -1    │
  └─────────────┴──────────┴──────────┘

  ⚠ 与普通模式核心差异：
     ① 所有阶段都扣精神（普通模式完全不扣）
     ② 成绩扣幅是普通模式的 2-3 倍
     ③ 从高一开始就有惩罚（普通模式高三才有）

─────────────────────────────────────
二、成绩增益缩水
─────────────────────────────────────
  所有正确选择获得的成绩水平增量 = max(0, 原始值 - 1)

  举例：
  ◆ 事件原本 +3 成绩 → 实际只 +2
  ◆ 事件原本 +1 成绩 → 实际 +0（被吃掉）
  ◆ 负增益不受影响（不会更负）
  ◆ 爱情/体能/关系等增益 → 不受影响

─────────────────────────────────────
三、假期门槛提高（getHolidayScoreReq）
─────────────────────────────────────
  触发"放假"格子进入假期的成绩要求：

  ┌───────────┬──────────┬──────────┐
  │  进度阶段  │ 普通门槛  │ 困难门槛  │
  ├───────────┼──────────┼──────────┤
  │  第2张图  │   60     │   70     │
  │  第3张图  │   100    │   110    │
  │  第5张图  │   130    │   140    │
  └───────────┴──────────┴──────────┘

  ⚠ 统一 +10 门槛，更容易错过假期！

─────────────────────────────────────
四、精神崩溃惩罚（applyHardSpiritPenalty）
─────────────────────────────────────
  触发条件：精神状态 < 45 时，每次移动额外触发

  惩罚机制：
  ◆ 从 5 个非精神状态中随机抽取 2 个
  ◆ 按当前数值高低加权，数值越高越容易被扣
  ◆ 每个被抽取的状态 -5 点

  抽中概率（按数值排序后）：
  ┌────────────┬──────────┬──────────────────────┐
  │   优先级    │   权重    │       说明            │
  ├────────────┼──────────┼──────────────────────┤
  │  第1高     │   50%    │ 最可能被扣            │
  │  第2高     │   20%    │                      │
  │  第3高     │   15%    │                      │
  │  第4高     │   10%    │                      │
  │  第5高     │    5%    │ 最不可能被扣          │
  └────────────┴──────────┴──────────────────────┘

  目标状态池（5个）：
     成绩水平、爱情进展、体能水平、学习状态、班级关系

  排除状态（1个）：
     精神状态（已是触发条件，不参与扣减）

  ⚠ 精神越低，越容易触发 → 越触发越崩 → 恶性循环！
  ⚠ 爱情进展也会被扣除，恋爱线不再安全！

─────────────────────────────────────
五、事件效果
─────────────────────────────────────
  事件基础效果与普通模式相同：
  ◆ 爱情进展正增益 × 班级关系倍率
  ◆ 波动范围：基础值 ±2（随机）
  ◆ 但成绩增益再额外 -1（见第二条）

─────────────────────────────────────
七、策略建议
─────────────────────────────────────
  ⚡ 优先维持精神 ≥45，避免触发崩溃惩罚
  ⚡ 成绩增益被削弱，需更精准选择正确选项
  ⚡ 假期门槛更高，必须更早规划成绩提升
  ⚡ 恋爱系统独立运作，可作为稳定优势线
  ⚡ 精神崩溃会连锁拖垮所有数值，恶性循环
